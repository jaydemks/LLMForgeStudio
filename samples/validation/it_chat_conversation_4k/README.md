# it_chat_conversation_4k (v4)

- Language: Italian (`it`)
- Rows: 4,000 JSONL prompt/response records
- Goal: chat-first baseline dataset with stronger variation and lower template repetition.

## Included
- Everyday conversational turns
- Practical writing assistance (email/chat/team contexts)
- Basic grammar guidance in Italian
- Short productivity/clarity tips

## Notes
This sample is designed for faster local validation versus larger corpora, while remaining diverse enough to reduce deterministic template collapse.
